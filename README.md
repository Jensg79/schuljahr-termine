# schuljahr-termine

Private Terminliste für den Schuljahresablauf (Fachleiter Mathematik, bayerisches Gymnasium).

## Dateien

| Datei | Zweck |
|---|---|
| `schuljahr-termine.md` | Terminliste im Format `YYYY-MM-DD \| Aufgabe \| Hinweis` |
| `task-erinnerung.md` | Dokumentation des täglichen Cowork-Tasks |

## Termine bearbeiten

Einfach `schuljahr-termine.md` direkt auf GitHub oder lokal bearbeiten.
Der Task liest beim nächsten Lauf automatisch die aktuelle Version.
